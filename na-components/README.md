# Nodes
SNN 형식의 IRIS Linear Regression 및 MNIST 분류에 대한 Node-red 노드 소스입니다.

## 폴더 분류
### 1. _python
IRIS, MNIST 에 대한 훈련 데이터와 분류 소스에 대한 폴더
### 2. -dataset
1개의 time-inject를 input으로 받아 1개의 무작위 IRIS 또는 MNIST 데이터셋을 output으로 내보내는 노드
### 3. -implement
dataset을 input으로 받아 trainning 데이터를 기반으로 분류한 결과값을 output으로 내보내는 노드
### 4. -print
implement를 input으로 받아 Debug/PkgMgr 형태로 변환하여 output으로 내보내는 노드
